Un buen editor de código es esencial para la productividad.

**Buena Práctica:** Usa **Visual Studio Code (VS Code)**. Es gratuito, ligero y tiene una gran cantidad de extensiones que facilitan el desarrollo con React.

1.  **Descarga e instala VS Code:** Ve a [https://code.visualstudio.com/](https://code.visualstudio.com/) y descarga la versión para tu sistema operativo.
2.  **Extensiones Recomendadas para VS Code (Opcional, pero muy útil):**
      * **ES7+ React/Redux/GraphQL/React-Native snippets:** Para atajos de código de React.
      * **Prettier - Code formatter:** Para formatear tu código automáticamente y mantener una **consistencia** en el estilo (buena práctica crucial).
      * **ESLint:** Para ayudarte a escribir código sin errores y seguir las **buenas prácticas** de JavaScript.

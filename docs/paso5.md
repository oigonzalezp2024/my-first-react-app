**Buena Práctica:** Divide tu interfaz de usuario (UI) en componentes **pequeños, independientes y reutilizables**. Cada componente debe tener una **única responsabilidad**.

  * **Componentes de Función:** Son la forma preferida y moderna de escribir componentes en React. Son funciones de JavaScript que retornan **JSX**.
  * **JSX (JavaScript XML):** Es una extensión de sintaxis que te permite escribir código similar a HTML directamente dentro de JavaScript. Se ve como HTML, pero es JavaScript real que React transforma en elementos de la UI.
  * **Archivos `.jsx`:** Aunque puedes usar `.js`, la convención `.jsx` es una **buena práctica** porque indica claramente que el archivo contiene código JSX, lo que mejora la legibilidad.
